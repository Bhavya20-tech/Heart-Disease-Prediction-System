# ❤️ Heart Disease Prediction System

A Machine Learning web application that predicts whether a patient is likely to have heart disease based on medical parameters.

## Features

- Predicts heart disease risk
- Interactive Streamlit interface
- Logistic Regression model
- Data preprocessing using StandardScaler
- Real-time prediction

## Technologies Used

- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy
- Joblib
- Matplotlib
  
## Dataset

UCI Heart Disease Dataset

## Model

- Logistic Regression
- Accuracy: 82%

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
